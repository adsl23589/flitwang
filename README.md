# 📺 YouTube 播放清單匯出工具 (YouTube Playlist Exporter)

這是一個專為「不懂程式碼的普通人」設計的超簡單小工具！
只要貼上 YouTube 播放清單的網址，就能一鍵將裡面所有影片的「標題」與「網址」匯出成文字檔 (`.txt`) 或 Excel 可讀的表格檔 (`.csv`)。

> 💡 **開源精神**：本工具基於 Python 撰寫，核心使用開源且知名的 `yt-dlp` 套件，安全無毒，完全免費。

## ✨ 核心功能

* **極簡視窗介面**：不需要在黑漆漆的視窗打程式碼，有按鈕可以直接點。
* **雙重格式匯出**：支援匯出成 `.txt` (純文字) 或 `.csv` (Excel 表格)。
* **免申請 API**：免去繁瑣的 Google API 申請流程，打開就能直接用。
* **自動防呆機制**：內建編碼處理 (`utf-8-sig`)，用 Excel 打開 CSV 也不會變亂碼。

## 📥 第一步：如何下載這個工具？

如果您不懂程式碼，請按照以下最簡單的下載方式：

1. 點擊這個 GitHub 網頁右上角的綠色按鈕 **`<> Code`**。
2. 在跳出的選單中，點擊 **`Download ZIP`**。
3. 將下載下來的 ZIP 壓縮檔「解壓縮」到您的電腦裡（例如放到桌面）。
4. 您會在資料夾裡找到一個名為 `youtube_exporter.py` 的檔案，這就是我們的主程式！

---

## 🚀 第二步：初次使用環境設定 (只需做一次)

如果您的電腦以前從來沒有跑過 Python 程式，請跟著以下步驟安裝「程式的引擎」。只要做過一次，以後就不需要再做了！

### 1. 安裝 Python 🐍
1. 前往 Python 官方下載頁面：https://www.python.org/downloads/
2. 點擊黃色的 **"Download Python 3.x.x"** 按鈕下載安裝檔。
3. ⚠️ **【極度重要】**：下載後執行安裝檔，在安裝視窗的最下方，**務必勾選 `Add Python to PATH`** (將 Python 加入環境變數)。
   > *備註：如果不勾選這個，後面的步驟您的電腦會認不得 Python 指令。*
4. 點擊 **"Install Now"** 直到安裝完成。

### 2. 安裝核心套件 (`yt-dlp`) 🛠️
1. 按下鍵盤上的 **`Windows 鍵` + `R`**。
2. 在彈出的執行框輸入 `cmd`，然後按 **Enter**，會出現一個黑底白字的視窗（命令提示字元）。
3. 複製下面這行指令，貼到那個黑色視窗裡，然後按 **Enter**：
   `pip install yt-dlp`

> 🔴 **常見錯誤：如果出現 `'pip' 不是內部或外部命令...`**
> 這代表電腦暫時找不到指令。請改用下面這行指令試試看：
> `python -m pip install yt-dlp`
> *(如果還是不行，請重新啟動電腦讓設定生效，再重試一次)*

4. 等待畫面跑完，最後看到 `Successfully installed...` 就代表安裝成功了，可以把黑視窗關掉。

## ▶️ 第三步：開始使用！

1. 打開您剛剛解壓縮的資料夾。
2. 找到 `youtube_exporter.py` 這個檔案。
3. 在檔案上按 **右鍵**，選擇 **"Edit with IDLE"** (這是安裝 Python 後內建的編輯器)。
4. 編輯器開啟後，按下鍵盤上的 **`F5`** 鍵，工具視窗就會彈出來囉！
   > *💡 提示：如果您想要更方便，可以直接對著該檔案 **點擊兩下左鍵**，通常也能直接開啟程式視窗。*

## ❓ 疑難排解 FAQ

### Q1: 按下匯出後，程式好像卡住了？
**A:** 請看視窗中間的狀態文字。如果顯示「正在讀取清單資訊」，代表程式正在背景努力工作中。如果您的播放清單有幾百部影片，可能需要等待幾十秒鐘，請耐心等待，視窗並沒有當機喔。

### Q2: 匯出的表格 (.csv) 檔，用 Excel 打開來還是亂碼怎麼辦？
**A:** 程式碼中已經設定了專門處理中文的格式。如果依然出現亂碼，請試著對 CSV 檔按右鍵 -> 選擇用「記事本」打開，確認裡面的文字是否正常顯示。正常的話也可以透過 Excel 的「資料 -> 從文字/CSV」功能來匯入。

### Q3: 程式跳出「發生錯誤：Private video」的訊息？
**A:** 這是因為您輸入的播放清單裡面，包含了設定為「私人 (Private)」的影片，或者是整個播放清單本身就是私人的。本程式只能抓取對外公開的資料，請確認該清單狀態至少是「不公開 (Unlisted)」或「公開 (Public)」。
